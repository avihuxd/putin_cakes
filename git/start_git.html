Welcome to GIT 2 fire 333

git init - יוצר תקייה של גיט שנוכל לעבוד איתו

git add - אורז את הקבצים שנרצה לשמור בבראנץ'/להכנה לשליחה לענן

git commit -m "message" - ואורז את החבילה עם הקבצים ושומר אותה עם חתימה בבראנץ/ מכין את החבילה לשיגור לענן

git branch - מציג רשימה של כל הבראנצ'ים/ענפים שקיימים בתיקייה

git branch branch_name - מייצר בראנצ חדש בשם שנרצה

git checkout branch_name - עובר לבראנצ שנתנו את השם שלו

git checkout -b branch_name - גם מייצר בראנצ חדש וגם עובר אליו
